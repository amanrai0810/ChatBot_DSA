## ⚙️ Setup Instructions

1. Clone the repository

```
git clone https://github.com/your-username/dsa-chatbot.git
```

2. Navigate to project folder

```
cd dsa-chatbot
```

3. Install dependencies (if backend exists)

```
npm install
```

4. Add your API key

```
const API_KEY = "your_api_key_here";
```

5. Run the project

```
npm start
```
